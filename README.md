# Serveur web avec un Arduino Nano

> Transforme ton Arduino Nano en petit serveur web local grâce à un shield Ethernet.

---

## Étape 1 — Installer le nécessaire

Pour pouvoir faire de ton Arduino Nano un serveur web, tu auras besoin de :

- Un **IDE Arduino** (ex: Arduino IDE 2.3.4)
- Un **câble USB** pour brancher l'Arduino à ton PC
- Un **shield Ethernet** compatible (ENC28J60 ou W5100/W5500)
